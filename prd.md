# Product Requirements Document

## Product Name

Working name: Local Property Website Studio

The final brand name can be changed before build. The website should present your service as a specialist website and lead-generation offer for local builders, landlords, rental businesses, and property managers.

## Goal

Create a professional landing website that helps you sell website packages to local builders and property managers. The site should communicate that you understand real estate operations, can build trust quickly, and can help clients get more leads, enquiries, bookings, and maintenance requests online.

## Target Customers

- Local builders and construction firms
- Property managers handling rental units
- Small real estate agencies
- Landlords with multiple properties
- Rental businesses that still depend on phone calls, WhatsApp, paper forms, or spreadsheets

## Core Problem

Many local builders and property managers have weak websites or no useful online system. Their current setup often misses leads, looks outdated, does not explain their services clearly, and forces customers to call or message manually for basic enquiries.

## Value Proposition

Modern websites and simple lead systems built specifically for local property businesses.

The website should show that you can deliver:

- A professional online presence
- Mobile-friendly pages
- Clear service and project presentation
- Lead enquiry forms
- WhatsApp/contact actions
- Property listing or project showcase sections
- Maintenance request forms for property managers
- Fast launch packages for small local businesses

## Primary Website Objective

Convert visitors into leads by getting them to:

- Request a free consultation
- Send a WhatsApp message
- Book a call
- Ask for a website package quote

## Secondary Objectives

- Build trust quickly
- Explain package options clearly
- Show sample website use cases
- Educate local business owners on why a better site helps them
- Make the service feel specialized, not generic

## Main Pages / Sections

### Home

The main landing page should include:

- Sticky navigation
- Hero section with clear offer
- Trust strip or short credibility metrics
- Services section
- Industry-specific use cases
- Website package/pricing section
- Process section
- Testimonials or placeholder case-study cards
- FAQ section
- Final contact CTA
- Footer

### Optional Future Pages

- Portfolio
- Pricing
- Contact
- Blog/resources
- Individual service pages

## Hero Requirements

The hero must immediately explain the offer.

Suggested headline:

"Websites That Bring More Leads to Local Property Businesses"

Suggested supporting copy:

"I build fast, professional websites for builders, landlords, and property managers, with enquiry forms, WhatsApp leads, project showcases, and simple tools that help you manage customer requests."

Primary CTA:

"Get a Free Website Plan"

Secondary CTA:

"View Packages"

## Services

The site should present service cards such as:

- Builder websites
- Property manager websites
- Rental enquiry landing pages
- Project and property showcase pages
- WhatsApp lead capture
- Maintenance request forms
- Google-ready local SEO setup

## Pricing / Packages

Initial package structure:

- Starter Website: single-page professional website for small businesses
- Growth Website: multi-section site with forms, project showcase, and WhatsApp lead capture
- Pro System: custom website with property/project listings, maintenance forms, and advanced enquiry flows

Exact prices can be added later. If pricing is not final, use "Starting from" or "Request quote".

## Functional Requirements

- Mobile responsive layout
- Navigation anchors
- Contact/lead form UI
- WhatsApp CTA link
- Clear package comparison
- FAQ accordion
- Clean footer with contact details
- Fast-loading images
- Accessible text contrast

## Non-Functional Requirements

- Fast page load
- Professional design suitable for local business owners
- Easy to edit content
- SEO-friendly headings and metadata
- No copied brand assets from the reference website
- Original copy, colors, imagery, and layout details

## Success Metrics

- Visitors understand the offer within 5 seconds
- Primary CTA visible above the fold
- Contact action available from every major section
- Site works well on mobile
- Business owner can understand packages without needing technical knowledge

## Out of Scope for First Version

- Full CRM
- User login
- Payment processing
- Client dashboard
- Automated property management software
- Complex backend

## First Build Recommendation

Build a polished static marketing site first. Add backend features only after validating demand with local builders and property managers.
