# Design Direction

## Overall Feel

The design should feel like a practical, premium service for local property businesses. It should be polished enough to build trust with builders and property managers, but not so abstract that it feels like a generic tech startup.

Keywords:

- Professional
- Local
- Clear
- Lead-focused
- Trustworthy
- Property-specific

## Reference Site Lessons

The reference site works well because it has:

- A clear hero message
- Strong CTA buttons
- Building imagery
- Floating metric cards
- Simple feature cards
- Pricing cards
- Testimonials
- Final CTA

For this project, reuse the structure but create a different brand, copy, colors, and visual identity.

Do not copy:

- Logo
- Brand name
- Purple-heavy palette
- Exact wording
- Exact testimonial text
- Exact pricing
- Exact layout details

## Brand Positioning

Suggested positioning:

"Websites and lead systems for local builders and property managers."

The design should make the visitor think:

- This person understands my type of business.
- This website service is not generic.
- I can get more enquiries from my local market.
- Starting looks simple.

## Visual Concept

Use a clean editorial-business layout with real estate imagery and subtle interface overlays.

Hero visual idea:

- A building/project image or website mockup
- Floating cards showing:
  - New enquiries
  - WhatsApp leads
  - Maintenance requests
  - Project views

This keeps the useful idea from the reference site while changing the business context.

## Color Palette

Recommended palette:

- Background: `#F7F4EF`
- Surface: `#FFFFFF`
- Primary text: `#1F2933`
- Secondary text: `#5E6673`
- Primary accent: `#176B5B`
- Accent dark: `#0F4D42`
- Warm accent: `#C47A3A`
- Border: `#DDD6CC`
- Soft green surface: `#E8F3EF`
- Soft copper surface: `#F7EDE3`

Use green as the main trust/operations color and copper as a construction/property accent.

## Typography

Recommended font direction:

- Modern sans-serif
- Strong H1
- Clear body copy
- Compact card titles

Possible fonts:

- Inter
- Manrope
- Plus Jakarta Sans

Use one font family for the first version unless a clear brand need appears.

## Layout Style

Desktop:

- Header at top with nav and CTA.
- Hero split between text and strong visual.
- Trust strip below hero.
- Service cards in a responsive grid.
- Packages in 3 cards.
- Process in horizontal steps or clean vertical timeline.

Mobile:

- Hero text first.
- Visual below.
- CTAs stacked or full-width.
- Cards in one column.
- Sticky or easy-to-reach contact action if practical.

## Component Style

Cards:

- White or lightly tinted surface
- Thin border
- 8px radius
- Minimal shadow
- Strong title
- Short description

Buttons:

- Primary green filled button
- Secondary white or outline button
- Copper accent only for highlights, not every CTA

Icons:

- Use simple line icons
- Keep icon backgrounds subtle
- Use consistent size

## Hero Copy Direction

Headline options:

- "Websites That Bring More Leads to Local Property Businesses"
- "Professional Websites for Builders and Property Managers"
- "Turn Local Property Interest Into Real Enquiries"

Supporting copy direction:

"I build fast, mobile-ready websites for builders, landlords, and property managers, with enquiry forms, WhatsApp leads, project showcases, and request flows designed for local customers."

## Section Copy Direction

Use simple, outcome-focused section titles:

- "Built for Property Businesses"
- "Everything Your Local Website Needs"
- "Packages for Every Stage"
- "From First Call to Launch"
- "Make It Easy for Customers to Contact You"

## Imagery Direction

Recommended image subjects:

- Finished homes or apartment buildings
- Local construction projects
- Property managers reviewing documents
- Modern building exteriors
- Website mockup on laptop/mobile

Avoid:

- Abstract tech dashboards as the only image
- Luxury real estate imagery that feels unrelated to local builders
- Dark images behind important text

## Trust Elements

Use trust indicators such as:

- Mobile-ready
- WhatsApp enabled
- Local SEO setup
- Fast launch
- Clear packages
- Built for builders and property managers

If real testimonials are not available, use honest proof alternatives:

- Sample project concepts
- Process clarity
- Package details
- Before/after website audit offer

## CTA Strategy

Primary CTA:

"Get a Free Website Plan"

Secondary CTA:

"View Packages"

Contact CTA:

"Message on WhatsApp"

Final CTA:

"Ready to get a better website for your property business?"

## Design Risks

- Looking too much like the reference website
- Making the offer too broad
- Using fake testimonials
- Making the site feel like software instead of a service
- Adding too many sections before the core offer is clear

## Design Checklist

- First viewport clearly says what is being sold.
- Target audience is visible immediately.
- CTA is visible without scrolling.
- Visuals relate to property/building businesses.
- Package cards are easy to compare.
- Mobile design is clean and readable.
- No copied reference branding or content.
